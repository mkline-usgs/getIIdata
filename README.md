getIIdata
=========

###Purpose
    Pulls II data from IRIS server

###Usage

######Run script setup 

    Run script for getIIdata.py module sets variables for getIIdata (runII.py):
```python
    # Initiate user variables 
    year = '2014'
    startday = '001'
    network = 'II'

    # Create object from getIIdata module
    obj = getIIdata.GetArgs(x, y, z)
```

######Basic execution

    How to run: ./runII.py
